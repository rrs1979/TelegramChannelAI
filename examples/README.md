# Configuration Examples

Ready-to-use templates for different channel types.

| File | Channel Type | Language |
|------|-------------|----------|
| `config_news_channel.py` | Ukrainian news with editorial voice | Ukrainian |
| `config_crypto_channel.py` | Crypto market analysis | English |
| `config_tech_channel.py` | Tech/AI news digest | English |

## Usage

1. Copy `CHANNEL_PROMPTS`, `SOURCE_CHANNELS`, and `EXCLUSIONS` into `channel_ai.py`
2. Adjust the system prompt to match your channel personality
3. Test with `python channel_ai.py` before enabling auto-publish
